# Library Management System (MERN)

This project is a basic library management web application developed for Avatar Meher Baba Pune Centre.

## Features
- Display all books in structured format
- Search books by name
- Backend API using Node.js & Express
- MongoDB database integration
- Simple and clean UI using React

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB Atlas

## Current Status
- Basic UI completed
- Book data is being collected and will be updated
- Search functionality working

## Future Improvements
- Category-wise filtering
- Admin panel for adding books
- Online ordering system (if needed)
- Stock management integration
